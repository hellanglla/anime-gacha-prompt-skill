# 配色体系与风格维度

## 核心色彩配方

主色调（必须融入正面提示词描述）：
```
black, crimson red, gold, ivory white
```

点缀色：
```
pink-white highlights
```

**规则**：任何角色的正面提示词都应以这四种颜色为基底，除非用户明确指定了其他配色方案。局部高光可用粉白提亮。

---

## 6维度风格控制

用于推导和微调角色的视觉风格。每个维度3档可选。

### 1. 华丽度 (Ornateness)

- **A. 简约**: 简约克制，干净利落，不过多装饰
- **B. 标准**: 华丽精致，有装饰元素但不过度
- **C. 奢华**: 极致奢华，巴洛克风格，金饰细节，复杂装饰

### 2. 色彩饱和度 (Saturation)

- **A. 低饱和**: 柔和的色彩，低饱和度，淡雅色调
- **B. 标准**: 色彩鲜明，饱和均衡
- **C. 高饱和**: 高饱和，鲜艳夺目，霓虹色彩

### 3. 气质倾向 (Temperament)

- **A. 神圣**: 神圣纯洁，天使般，纯净无瑕，圣光笼罩
- **B. 优雅**: 高贵优雅，端庄精致，贵族气质
- **C. 魅惑**: 魅惑诱人，性感撩人，摄人心魄
- **D. 威严**: 威严霸气，压迫感，支配气场
- **E. 阴暗**: 神秘阴暗，暗黑诡异，阴影笼罩

### 4. 细节复杂度 (Detail)

- **A. 简洁**: 细节简洁，线条干净
- **B. 标准**: 精致细节但可控，复杂度均衡
- **C. 复杂**: 极致细节，高度精细，图案密集

### 5. 特效强度 (Effects)

- **A. 无特效**: 无光效，自然光源
- **B. 轻微**: 轻微辉光，柔和粒子，柔和光效
- **C. 强烈**: 强烈光芒，戏剧化特效，魔法光环，能量粒子

### 6. 渲染风格 (Rendering)

- **A. 原始**: 锐利细节，清晰边缘，硬朗光影，强对比，真实肤质，清晰背景，鲜艳色彩
- **B. 精修**: 柔和辉光，柔光照明，漫射光，光滑皮肤，柔和高光，边缘光，氛围粒子，景深虚化，背景模糊，平衡色彩，柔色调，商业级精修质感
