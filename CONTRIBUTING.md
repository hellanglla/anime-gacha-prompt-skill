# 贡献指南

感谢你对本项目的关注！

## 可以贡献的内容

### 1. 报告姿势生成问题

如果你发现某个姿势生成不准确（如"侧躺"被画成"趴地"），请提交 Issue：
- 使用的AI平台（GPT-4o/Midjourney/Stable Diffusion）
- 输入的角色描述和选择的姿势
- 实际输出 vs 预期输出

### 2. 扩展姿势库

在 `references/poses.md` 中添加新姿势：
- 使用纯中文空间关系描述
- 对易混淆姿势增加"排除歧义"说明
- 禁止添加英文tag

### 3. 提交生成示例

在 `examples/` 目录下添加：
- 角色名 + 使用的提示词版本
- 完整的正面提示词文本
- 说明使用的AI平台和关键参数

### 4. 改进 Skill 核心

修改 `SKILL.md` 时请注意：
- 保持本体在 200 行以内
- 详细内容应拆分到 `references/` 而非直接内联
- 保持5段式输出格式不变

## Commit 规范

- `feat:` 新功能
- `fix:` 修复bug
- `docs:` 文档更新
- `refactor:` 重构

## ⚠️ 注意事项

- ❌ 不接受包含暴力、色情、政治敏感内容的贡献
- ✅ 姿势描述必须使用纯中文，禁止英文tag
